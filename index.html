MUITO PRAZER, SOU SAMUEL MARTINS.

Sou um desenvolvedor Backend especializado em Java e Spring. Minhas habilidades abrangem desde o desenvolvimento de microserviços até a implementação de APIs RESTful, utilizando tecnologias como Spring Boot, MySQL e RabbitMQ. Tenho conhecimentos sólidos em Java, HTML, CSS e Javascript, proporcionando uma base robusta para o desenvolvimento de aplicações web.

Além disso, possuo experiência em integrações complexas, como o projeto de um chatbot para WhatsApp que utiliza Java, Spring Boot, Twilio e Finnhub para fornecer informações sobre cotações de ações de maneira ágil e conveniente.

Com uma formação em Engenharia de Software em andamento, estou comprometido em aprimorar minhas habilidades e contribuir significativamente para projetos desafiadores.

MEUS PROJETOS

Controle de Gastos

Este projeto é uma API RESTful desenvolvida em Java utilizando o framework Spring Boot e tecnologias relacionadas. A finalidade da aplicação é auxiliar o usuário no controle de suas despesas, permitindo o gerenciamento por categorias e status de pagamento.

ChatBotWhatsapp

Um chatbot para WhatsApp que fornece informações básicas sobre cotações de ações do mercado financeiro. O chatbot recebe símbolos de cotação válidos (código da ação) através de mensagens e responde com informações sobre a ação correspondente.

Microserviço de envio de e-mails.

Criação e implementação de um microserviço robusto de envio de e-mails. O objetivo é garantir a entrega eficiente de e-mails, permitindo comunicação síncrona e assíncrona para maior flexibilidade e escalabilidade.
